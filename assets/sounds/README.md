# Sound Files

Add MP3 sound files to this folder. The app will look for these files:

## Required Sounds

1. `beep.mp3` - Simple beep for proximity detection (high pitch)
2. `killed_kenny.mp3` - "Oh my God, they killed Kenny!"
3. `kenny_crying.mp3` - Kenny crying sound
4. `kenny_muffled.mp3` - Kenny's muffled voice
5. `omg_kenny.mp3` - "OMG Kenny!"
6. `kenny_dies.mp3` - Kenny death sound
7. `woah_kenny.mp3` - "WOAH KENNY!"

## Where to Download

Visit: https://www.myinstants.com/en/search/?name=kenny

Look for South Park related Kenny sounds and download them.

## Adding New Sounds

1. Add the MP3 file to this folder
2. Edit `assets/config/sounds.json` to add the new sound entry
3. Rebuild the app

## Note

Make sure sound files are properly licensed for your use case.
